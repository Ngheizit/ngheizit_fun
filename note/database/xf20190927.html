& 配置ArcGIS Runtime SDK for Android

0. 将项目Gradle存储库（Module:app）更新为Esri的Bintray Mavrn存储库URL

allprojects {
    repositories {
        ...
        maven {
            url 'https://esri.bintray.com/arcgis'
        }
    }
}

1. 更新应用程序模块Gradle依赖项（Project:xxx），以包括适用于Android的ArcGIS Runtime SDK依赖项

dependencies {
    ...
    implementation 'com.esri.arcgisruntime:arcgis-android:100.6.0'
}

2. 添加Java 8编译选项（Project:xxx）

android {
    ...
    compileOptions{
        sourceCompatibility JavaVersion.VERSION_1_8
        targetCompatibility JavaVersion.VERSION_1_8
    }
}

3. 在清单文件（AndroidManifest）中设置权限运行网络访问并指示应用程序使用OpenGL 2.0或更高版本

<uses-permission android:name="android.permission.INTERNET" />
<uses-feature android:glEsVersion="0x00020000" android:required="true" />

4. 总结

参考网址：ArcGIS Developers；强烈建议使用 Android Studio 3 版本以上；
