& SQL关系型数据库标准语言 - 数据类型

数据类型及对应含义

// 长度为n的定长字符串
CHAR(n)
CHARCTER(n)

// 最大长度为n的变长字符串
VARCHAR(n)
CHARCTERVARTING(n)

// 字符串大对象
CLOB

// 二进制大对象
BLOB

// 长整型 - 4字节
INT
INTEGER

// 短整型 - 2字节
SMALLINT

// 大整形 - 8字节
BIGINT

// 定点数，由p位数字（不包括符号、小数点）组成，小数后面有d位数字
NUMERIC(p, d)

// 同NUMERIC
DECIMAL(p, d)
DEC(p, d)

// 取决于机器精度的单精度浮点型
REAL

// 取决于机器精度的双精度浮点型
DOUBLE PRECISION

// 可选精度的浮点数，精度至少为n为数字
FLOAT(n)

// 逻辑布尔量
BOOLEAN

// 日期，包含年、月、日，格式为YYYY-MM-DD
DATE

// 时间，包含一日的时、分、秒，格式为HH:MM:SS
TIME

// 时间戳类型
TIMESTAMP

// 时间间隔类型
INTERVAL
