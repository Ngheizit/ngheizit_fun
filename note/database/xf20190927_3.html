& ArcGIS Engine开发常见问题总结

0. 绑定ArcGIS Engine产品代码

需在项目文件Program.cs中的Main()方法语句下的Application.Run(...)前添加下述代码：

ESRI.ArcGIS.RuntimeManager.Bind(ESRI.ArcGIS.ProductCode.Engine);

1. 项目相对位置定位(...\bin\Debug\)

首相定位到项目调试的exe文件，然后再获得该文件所在的目录以实现项目目录定位

string exe = System.Diagnostics.Process.GetCurrentProcess().MainModule.FileName;
string PATH_ENVIMENT = System.IO.Path.GetDirectoryName(exe);

2. 需注意 ESRI.ArcGIS.Carto 和 ESRI.ArcGIS.Geodatabase 的ILayer接口区别

同时加到项目引用中
